# Iris-Based Patient Identification Project

## Описание проекта

Проект посвящен задаче идентификации пациентов по радужной оболочке глаза. Основная цель — создать модель, способную точно классифицировать изображения радужной оболочки и идентифицировать пациента среди множества классов.

## Бизнес-задача

Обеспечить надежную идентификацию пациентов в медицинских учреждениях с использованием изображений радужной оболочки, минимизируя ошибки ложного допуска (FAR) и ложного недопуска (FRR).

## Тип задачи

Классификация изображений с возможностью последующего дообучения при увеличении количества классов.

## Данные

* Датасет содержит фотографии глаз 89 пациентов клиники.
* Каждый пациент имеет от 3 до 8 изображений.
* Изображения организованы по папкам, размеченным уникальными идентификаторами пациентов (1–89).
* Входные данные: изображение переднего сегмента глаза.

## Требования к точности

* FRR (False Rejection Rate, ложный недопуск) ≤ 3%
* FAR (False Acceptance Rate, ложный допуск) ≤ 2%
* Идентификация: точность ≥ 98%
* Если вероятность предсказания класса < 97%, применяется недопуск (пациент отсутствует в базе).

## Модель

* Архитектура основана на сверточных нейронных сетях (CNN).
* На 89 классах удалось достичь accuracy 0.98 на тестовой выборке.
* Обучение: 350 эпох, общее время — около 1,5 часов.
* Возможное улучшение: классификация каждого глаза отдельно для увеличения точности.


## Статус проекта: Завершен.
